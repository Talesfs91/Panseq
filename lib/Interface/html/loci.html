Loci Finder

The Loci Finder constructs loci sets that are maximized with respect to the unique number of fingerprints, as well as the discriminatory power of the loci among the input dataset. Input is a tab-delimited file with row names identifying the loci, and column names identifying the strains or genomes the loci are found in. Missing data may be denoted by the characters '?', '-', or '.'
